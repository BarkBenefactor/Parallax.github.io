AS Val 9x39

3D Game-Ready Assets

This project was started to test my ability to create not only a Realistic Game-Ready model of my favorite rifle, but to then create a more stylized version of the same weapon. Inspiration for the stylized version comes from Facepunch Std's RUST.

 
  • Designer: Brandon Shrader
  • Skills: 3D Modeling, Texturing, & Concepting
  • Tools: Blender 3D & Substance Painter
 

Project Details

Realistic ASVal: High Poly: 32,0000. Low Poly: 13,000 Poly, 1 PBR Material


Stylized ASVal: 5,300 Poly, 1 PBR Material


Both of these props are intended, and created to be game-ready assets. Easily rigged, and ready to function within any engine from Unity to Unreal Engine.
